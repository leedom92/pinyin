# ✍️ A Chinese Characters Toy. 一个汉字拼音小玩具。

The toy is based on [pinyin](https://github.com/hotoo/pinyin).

## Preview
[Click here](https://pinyin.leedom.me)

## Install
```sh
git clone https://github.com/leedom92/pinyin.git

pnpm install

pnpm dev
```
